KF-21 비행제어 시스템/소프트웨어 OFP 통합 검증

📡 담당 CSU 및 주요 업무

  • 아날로그/디스크리트 신호 수집 및 검증 - 다양한 센서(조종사 입력) 신호 검증
  • 작동기 관리 - 제어 신호 출력 및 피드백 값 모니터링 검증
  • IMU 모니터 관리 - 관성측정장치 디지털 데이터 모니터링 및 검증
  • NVM 데이터 관리 - 비휘발성 영역 Block Size 검증

🔧 통합 테스트 시험 환경 구축 및 In-house Tool 활용한 자동화 검증

  • 절차서 기반의 자동화 시험 프로그램 개발
  • 통합 테스트 시험 환경(ETS)를 활용한 아날로그, 디지털, 피드백 값 등을 In-house Tool을 이용해 제어
  • 다양한 입출력 신호를 자동화 스크립트를 적용하여 실시간(ms)으로 수집 및 검증

🎯 실시간 비행데이터 시각화 프로그램 개발

  • 비행제어 시스템에서 수집된 데이터를 python 기반 시각화
  • ex) NVM 데이터 또는 MFD Display 값 수집 및 데이터 그래프화 (using pytesseract)
